Use the base ISO and exposure rules on purpose.

S-Log basics

  • Low base ISO: 800
  • High base ISO: 12,800
  • Use manual white balance
  • Avoid underexposure

Best practice

  • Expose slightly to the right by about +1 to +2
  • Do not flash all the way into clipping
  • Use ND filters outdoors if needed

This one is built like a clean technical walkthrough so learners make one setting decision at a time.

Guided flow

Quick cheat sheet

🌡️ Set Kelvin manually 💡 3200K = tungsten-ish ☀️ 5600K = daylight-ish 🎥 180° shutter = natural blur 🤝 Match Kelvin with your partner
1
White balance basics
Manual white balance first, not auto.
Start here

How to set it

  • Menu → Exposure/Color → White Balance
  • Select C.Temp/Filter
  • Adjust Kelvin manually

Reference values

  • 3200K for tungsten-type light
  • 5600K for daylight-type light
  • Match Kelvin with the other shooter if working together
